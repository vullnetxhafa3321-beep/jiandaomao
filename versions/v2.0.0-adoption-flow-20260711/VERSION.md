## 相对 v1 本版本新增 / 调整

1. **领养贴**：绝育 / 性别（女孩男孩）/ 约 x 岁 / 健康标签 / 疫苗；描述 + 照片发布
2. **送医流程**：带猫去医院卡片含「已就医 · 去发布领养」，直达领养发布页
3. **流浪发现**：仅「要不起」求助；去掉同步到待领养；评论可上传新图片
4. **附近友好医院**：统一附近列表 + 折扣说明；滴滴入口合并为「叫滴滴宠物专车」
5. **论坛安全**：约 1km 模糊定位、联系方式、已救助网名与首页喜报
6. **救助站**：地图/列表心形小屋 PNG 图标
